Section comment
============
2020-07-21 -> 2021-03-19


A **section comment** is a comment that looks like this:


```php

<?php

//--------------------------------------------
// MY SECTION
//--------------------------------------------

```


**Section comments** can be used to separate sections.


```txt

//--------------------------------------------
// SECTION ONE
//--------------------------------------------

This is the content of section one.
This is the content of section one.
This is the content of section one.

//--------------------------------------------
// SECTION TWO
//--------------------------------------------

This is the content of section two.
This is the content of section two.
This is the content of section two.


```
